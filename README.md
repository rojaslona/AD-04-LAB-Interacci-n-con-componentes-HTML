<h2>🔔 Tarea 1</h2>

“ES HORA DE UNA HAMBURGUESA”

<h2>🕵️‍♂️ Tarea 2</h2>

”¡Alguien hizo clic en BURGER TOWN!”

⸻

<h1>📌 Escenario 2</h1>

<h2>🍔 Tarea 3</h2>

”¡Una imagen de una HAMBURGUESA!”
